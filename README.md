# simple-backend-file-system

This is a beginner backend project created to understand how backend systems work and how servers handle user input.

The application allows users to create tasks through a form. Each task contains a title and description, and when submitted, a file is created on the server using the task title as the filename. Users can also read the saved files through the application.

## Tech Stack
- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- Tailwind CSS
- Vanilla JavaScript

## Features
- Create tasks using a form
- Save tasks as files on the server
- Read saved task files
- Simple UI styled with Tailwind CSS
- Server-side rendering using EJS

## Purpose
This project was built for learning purposes to understand:
- Backend fundamentals
- File system operations
- Handling form data
- Server-side rendering
- Basic full-stack workflow
